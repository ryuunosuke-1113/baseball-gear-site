# Baseball Gear Catalog（野球用品カタログ）

静的JSONデータで管理する、野球用品の商品紹介サイトです。  
一覧表示・検索・カテゴリ/ブランド絞り込み・詳細ページに対応しています。

## Demo
- ローカル起動：`npm run dev`
- （公開URLができたらここに貼る）

## Features
- 商品一覧（カード表示）
- 検索（キーワード）
- 絞り込み（カテゴリ / ブランド）
- 商品詳細ページ（/products/[id]）

## Tech Stack
- Next.js (App Router)
- TypeScript
- JSON data (`data/products.json`)

## Setup
```bash
npm install
npm run dev
